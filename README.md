# DataChallenge
Proyecto para microeconometría

## Contenido del repositorio
### Bases de datos Airbnb:
1) listings
2) listings_scrapped <- de ahí salió la variable estimated_revenue, que se usa al final del análisis
3) reviews

### Bases de datos sobre la CDMX:
1) neighbourhoods.geojson
2) ICMM <- ingresos estimados por municipio
  2.1) mun_keys_ingresos <- dataset asociado para identificar claves de municipios
3) Municipal delitos

### Codigos:
1) topic_shares: Jupyter notebook calculando shares por topic en reviews
2) mainregs.R: código procesando datasets y obteniendo modelos principales
